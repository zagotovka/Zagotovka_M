# Проект: Zagotovka
- **Плата:** NUCLEO-F767ZI
- **Чип:** STM32F767ZIT6 (Cortex-M7)
- **Память:** 2MB Flash, 512KB SRAM
- **Интерфейсы:** Ethernet (LAN8742A), USB FS, ST-LINK V2-1

# Распиновка Nucleo-144:
- LD1 (Green): PB0
- LD2 (Blue): PB7
- LD3 (Red): PB14
- User Button: PC13
- Debug Console: UART3 (PD8/PD9)
